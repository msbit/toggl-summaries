# Toggle Summaries

## Toggl Sessions

    toggl_sessions.rb --database <database> \
                      --since <since-date> \
                      --until <until-date> \
                      --workspace <workspace-name> \
                      [--[no-]billable] \
                      [--client <client-name>] \
                      [--project <project-name>] \
                      [--tag <tag-name>] \
                      [--task <task-name>]

## Toggl Totals

    toggl_totals.rb --since <since-date> \
                    --until <until-date> \
                    --workspace <workspace-name> \
                    [--[no-]billable] \
                    [--client <client-name>] \
                    [--project <project-name>] \
                    [--tag <tag-name>] \
                    [--task <task-name>]
